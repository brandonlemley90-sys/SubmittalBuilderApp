# User Installation & Update Guide

Welcome to the Submittal Builder! This guide will help you install and keep your application up to date.

## 1. Initial Installation
Your developer will provide you with a file named `Setup_DenierAI_Submittal_Builder.exe`.

### Steps:
1. Save the file to a folder on your computer (e.g., `C:\SubmittalBuilder`).
2. **Right-click** in the folder and select **"Open in Terminal"** (or "Open PowerShell window here").
3. Type the following command and press **Enter**:
   ```powershell
   .\Setup_DenierAI_Submittal_Builder.exe
   ```
4. The installer will download the latest files and create a shortcut on your desktop.

---

## 2. Running the Application
Once installed, you can launch the app from the Desktop shortcut. The application runs in your browser at:
`http://127.0.0.1:5002`

---

## 3. Updating the Application
When a new version is released, you will see a **Blue Banner** at the top of your login screen.

### How to Update:
1. Click the **"Update Now"** button on the banner.
2. The button will change to **"Downloading..."**.
3. Once the download is complete, the application will **automatically close and restart**.
4. After restarting, a **Green Banner** will confirm: "Update Installed Successfully!".

## Troubleshooting
- **Port Conflict**: If the app fails to open, ensure no other programs are using port 5002.
- **Red Warning**: If the banner shows an error, check your internet connection.
