# User Access Guide

Welcome to the Submittal Builder! This application is now a centralized web service. There is no longer a need to install anything on your personal machine.

---

## 1. Accessing the Application

You can access the Submittal Builder from any modern web browser at:
**[https://submittalbuilderapp.onrender.com/](https://submittalbuilderapp.onrender.com/)**

---

## 2. Using the Application

1. **Login**: Use your provided credentials to sign in.
2. **Project Dashboard**: View your current jobs and their statuses.
3. **New Submission**: Upload your Excel specs, PDF drawings, and other required documents.
4. **Download Results**: Once processing is complete, you can download your final submittal package directly from the dashboard.

---

## 3. Troubleshooting

- **Access Denied**: Ensure you are using the correct email and password.
- **Processing Time**: Large PDF files may take several minutes to process. Check the "Status" column on your dashboard for live updates.
- **Connection Issues**: If the page fails to load, check your internet connection and try refreshing the browser.
- **Browser Support**: We recommend using the latest version of Google Chrome or Microsoft Edge for the best experience.
